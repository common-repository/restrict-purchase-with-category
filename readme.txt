== Description ==

This Plugin performs following functions. Developed by <a href="https://www.themelocation.com/">themelocation</a>

1- If Product Added from Category A then Other Categories Products will not be added<br>
2- It limits purchases of products to just one Category.

Main function of this plugin is once you added a product from One category; You can not purchase product from any other category.<br> This Plugin Works Well for Delivery business and Restaurants.

This plugin works on All Store Categories once you have installed and activated. This plugin by default enabled the functionality.<br>

If you have any Questions, <a href="https://www.themelocation.com/contact/">Contact Plugin Support</a><br>

== Installation ==

1. From the Dashboard:
1. Plugins
1. Add New
1. Search "Category Restriction WooCommerce"
1. Install Now
1. Activate
1. Go to Settings -> Category Restriction WooCommerce
